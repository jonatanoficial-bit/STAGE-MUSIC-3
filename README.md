# Stage Music v4.0.0

Aplicativo gratuito para cifras, letras, repertórios e Sala Live para equipes de música.

## Destaques da Fase 33

- Página de instalação pelo navegador para Android, iPhone, tablet e computador.
- Página de feedback de teste em massa.
- Feedback pode ser copiado, compartilhado e salvo localmente.
- Quando o usuário está com Google ativo, o feedback também tenta salvar em `users/{uid}/testFeedback`.
- Home e roteiro de teste agora orientam a equipe para instalar, testar e enviar retorno.
- Firebase, Sala Live, Catálogo Global e regras foram preservados.

## Uso

Abra `index.html` ou publique a pasta na raiz do GitHub Pages.
