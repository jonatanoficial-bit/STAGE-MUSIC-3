
## Stage Music v4.1.0 — Fase 34

Entrada por convite perfeita para banda e igreja.

- Nova página `convite.html` para quem recebeu link ou código da Sala Live.
- Fluxo direto por função: Vou cantar, Vou tocar ou Sou diretor.
- Perfil de convidado salvo no navegador para reduzir cliques nos próximos testes.
- Sala Live mostra escolha rápida de função antes de abrir cifra/letra.
- Quando a sala está na nuvem, o app explica o login Google sem jogar o usuário para fora de forma brusca.
- Play Store e App Store ainda não são prometidas: a comunicação oficial passa a reforçar uso pelo navegador em qualquer celular.

Firebase preservado: `js/firebase-config.js`, `js/firebase-runtime.js`, `js/firebase-live.js`, `js/cloud-sync.js` e `firestore.rules` não foram alterados.

# Stage Music v4.0.0

Aplicativo gratuito para cifras, letras, repertórios e Sala Live para equipes de música.

## Destaques da Fase 33

- Página de instalação pelo navegador para Android, iPhone, tablet e computador.
- Página de feedback de teste em massa.
- Feedback pode ser copiado, compartilhado e salvo localmente.
- Quando o usuário está com Google ativo, o feedback também tenta salvar em `users/{uid}/testFeedback`.
- Home e roteiro de teste agora orientam a equipe para instalar, testar e enviar retorno.
- Firebase, Sala Live, Catálogo Global e regras foram preservados.

## Uso

Abra `index.html` ou publique a pasta na raiz do GitHub Pages.
