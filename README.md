# Stage Music

**Build atual:** v1.5.0  
**Fase:** 10 — Home Premium AAA e Refinamento Visual Global  
**Gerada em:** 10/06/2026 às 10:24 (America/Sao_Paulo)

Aplicativo PWA mobile-first para cifras, repertórios e operação profissional ao vivo, com foco em palco, ensaio e organização musical.

## Destaques da build
- nova Home premium com visual mais impactante e mais coerente com um produto comercial;
- entrada rápida para Modo Live, Biblioteca, Editor e Repertórios;
- dashboard inicial com dados reais da biblioteca e dos setlists locais;
- resgate de repertórios recentes e cifras recentes diretamente na tela inicial;
- pesquisa iniciada na Home já segue para a Biblioteca;
- sistema anti-break v9 mantido;
- funcionamento local e offline preservado.

## Executar
Abra com servidor local, por exemplo:

`npm start`

## Validar a build

`npm run check`

## Hospedagem gratuita
Compatível com GitHub Pages, Vercel estático e Firebase Hosting. A estratégia de custo zero está documentada em `docs/ESTRATEGIA-CUSTO-ZERO.md`.
