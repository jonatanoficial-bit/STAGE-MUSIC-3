# Testes recomendados — Fase 35

1. Diretor cria Sala Live conectado.
2. Vocalista entra pelo convite e visualiza letra.
3. Músico entra pelo convite e visualiza cifra.
4. Diretor envia aviso rápido duas vezes; o app deve bloquear duplicação acidental.
5. Desligar internet por alguns segundos; a leitura deve continuar no aparelho.
6. Ligar internet; a sincronização deve tentar recuperar automaticamente.
7. Trocar música rapidamente; não deve saltar duas músicas por toque duplicado.
