# Release Candidate Final — Stage Music v4.5.0

Esta versão consolida as fases de produto, Sala Live, leitura por função, estabilidade, instalação pelo navegador, feedback e internacionalização.

## Situação de distribuição

O Stage Music ainda não está na Play Store nem na App Store. A distribuição oficial desta build é via navegador/GitHub Pages, compatível com Android, iPhone, tablet e computador.

## Caminho de homologação

Use `homologacao.html` antes de divulgar a versão para grupos maiores.
