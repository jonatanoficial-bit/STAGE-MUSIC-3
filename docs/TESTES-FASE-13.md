# Testes — Fase 13

1. Criar sala a partir de repertório existente.
2. Copiar o código da sala.
3. Abrir outra aba e entrar com o código.
4. Avançar e voltar músicas como diretor.
5. Confirmar atualização da posição na aba do músico.
6. Enviar comandos rápidos.
7. Abrir o Modo Live conectado.
8. Encerrar e sair da sala.
9. Executar `npm run audit`.
