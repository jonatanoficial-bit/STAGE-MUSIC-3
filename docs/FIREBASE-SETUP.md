# Firebase Setup — Stage Music

## Modo gratuito recomendado
Use o plano **Spark** do Firebase para começar sem custo.

## Serviços necessários
- Authentication
- Cloud Firestore

## Passos
1. Crie um projeto no console Firebase.
2. Adicione um app Web.
3. Copie as chaves do SDK.
4. Cole as chaves em `js/firebase-config.js`.
5. Ative o provedor Google e/ou e-mail/senha no Authentication.
6. Ative o Firestore em modo de produção e ajuste as regras do seu projeto.

## Observação
Se o Firebase não estiver configurado, o Stage Music continuará totalmente utilizável em modo local/offline.
