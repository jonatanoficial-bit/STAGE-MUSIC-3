# Catálogo Global Administrado

## Como publicar uma cifra global
1. Entre com a conta Google `jonatanoficial@gmail.com`.
2. Abra **Inserir cifra**.
3. Preencha título, artista, tom, BPM e conteúdo.
4. Localize o painel **Catálogo global**.
5. Toque em **Publicar no catálogo global**.
6. Abra **Buscar cifra** e selecione **Catálogo global**.

## Como atualizar
1. Abra **Buscar cifra**.
2. Filtre por **Catálogo global**.
3. Toque em **Editar global**.
4. Faça a alteração.
5. Toque novamente em **Publicar no catálogo global**.

## Como remover
1. Abra uma cifra global em **Editar global**.
2. Toque em **Remover do catálogo**.
3. Confirme a ação.

## O que os usuários comuns podem fazer
- visualizar cifras globais;
- favoritar;
- adicionar diretamente a repertórios;
- usar no Modo Live;
- criar uma cópia privada para editar.

Usuários comuns não podem alterar nem excluir a versão global.

## Regras do Firestore
Depois de publicar esta build no GitHub, copie o arquivo `firestore.rules` para:

`Firebase Console → Firestore Database → Regras → Publicar`

Sem essa publicação, o aplicativo não conseguirá gravar em `globalSongs`.
