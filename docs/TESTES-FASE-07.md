# Testes — Fase 7

1. Abra um repertório e inicie o Live.
2. Teste Rolar/Pausar e velocidades 1 a 10.
3. Use os botões de parte e as setas para cima/baixo.
4. Ative Repetir parte e confirme o retorno seguro.
5. Recarregue a página e confirme recuperação da posição.
6. Teste em orientação vertical e horizontal.
7. Execute `npm run audit`.
