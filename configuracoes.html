Configuração gratuita

Como ativar o Firebase sem gastar

  1. Crie um projeto no Firebase no plano gratuito Spark.
  2. Ative Authentication e Firestore.
  3. Copie as chaves para js/firebase-config.js.
  4. Publique a build no GitHub Pages ou Firebase Hosting.
  5. Faça login online e use os botões de enviar/baixar.
