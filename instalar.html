USO PELO NAVEGADOR

Stage Music funciona no celular, tablet e computador.

Por enquanto não é necessário baixar pela Play Store ou App Store. Basta abrir o link no navegador. Quem quiser pode adicionar o app à tela inicial para abrir como aplicativo.

Entrar em uma Sala Live Voltar ao início
Android / Chrome
  1. Abra o link do Stage Music no Chrome.
  2. Toque no menu de três pontos.
  3. Escolha Adicionar à tela inicial ou Instalar app.
  4. Abra pelo ícone criado.
iPhone / Safari
  1. Abra o link no Safari.
  2. Toque no botão de compartilhar.
  3. Escolha Adicionar à Tela de Início.
  4. Abra pelo ícone criado.
Computador
  1. Abra o link no Chrome, Edge ou outro navegador moderno.
  2. Entre com Google para salvar repertórios e cifras.
  3. Use o computador como direção musical ou para preparar listas.

Importante para o teste

Use uma janela normal do navegador, não anônima. A janela anônima pode apagar o login e dados temporários quando for fechada.
